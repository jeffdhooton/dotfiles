# Jeff's Dotfiles

dotfiles for neovim

## Requirements

- Have [neovim](https://neovim.io/) installed.
- Mac OS

## Installation

1. Clone this directory down to `~/.config/nvim`
2. Open that directory with nvim `nvim ~/.config/nvim`
3. Run `:PlugInstall`

## Plugins

### Color
Dracula

### Completion
Coc

### Status Bar
vim-airline

### Git integration
vim-signify
vim-fugitive
vim-rhubarb

### Fuzzy Find
